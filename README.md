# حصن المسلم 🕌

موقع شامل للأدعية والأذكار الإسلامية من الكتاب والسنة النبوية الشريفة.

## المميزات ✨

- 🎯 واجهة سهلة وجميلة
- 🔍 محرك بحث قوي
- 📱 تصميم متجاوب (يعمل على جميع الأجهزة)
- 📋 أقسام منظمة (صباح، مساء، طعام، نوم، صلاة، سفر)
- 📋 نسخ والمشاركة سهلة
- 🌙 دعم اللغة العربية كاملاً

## الأقسام الرئيسية 📚

1. **أذكار الصباح** - أذكار وأدعية الصباح
2. **أذكار المساء** - أذكار وأدعية المساء
3. **قبل وبعد الطعام** - الأدعية المتعلقة بالطعام
4. **أذكار النوم** - أدعية قبل النوم
5. **أدعية الصلاة** - الأدعية المتعلقة بالصلاة
6. **أدعية السفر** - أدعية المسافر

## كيفية الاستخدام 🚀

1. افتح الموقع في المتصفح
2. اختر قسماً من الأقسام الرئيسية
3. أو استخدم خاصية البحث للبحث عن دعاء معين
4. انقر على "نسخ" لنسخ الدعاء
5. انقر على "شارك" لمشاركة مع الآخرين
